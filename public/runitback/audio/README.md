# Run It Back — Soundtrack

Current tracklist (played in order, looping):

1. `david-dallas-running.mp3` — David Dallas, "Running"
2. `empire-of-the-sun-alive.mp3` — Empire of the Sun, "Alive"
3. `john-newman-love-me-again.mp3` — John Newman, "Love Me Again"

The player autoplays muted on page load (browsers block unmuted autoplay)
and unmutes automatically on the visitor's first click/tap/keypress.

To add or change tracks, drop MP3 files in this folder and update the
`TRACKS` array in `components/runitback/MusicPlayer.tsx`.
